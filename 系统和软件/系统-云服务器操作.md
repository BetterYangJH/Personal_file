## 连接

*   命令行连接：ssh username@IP
*   MobaXterm连接：配置好密钥，免密码

## 操作

上传本地文件

*   scp -P 端口 c://xxxx.txt user@ip:/home/root
*   scp -r localfile.txt username@192.168.0.1:/home/username/

图形化界面

*   更新软件库apt-get update
*   升级软件apt-get upgrade
*   安装ubuntu桌面系统apt-get install ubuntu-desktop
*   重启服务器reboot
