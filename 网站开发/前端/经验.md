# JS

*   ajax

    *   ajax和form天然跨域

    *   动态加载：只加载一小块，方便从不同地方获取数据，且省流量，设计好参数，第一个参数表明状态，第二个表明位置

        ```js
        $.ajax{
            Type:post,
            contentType:application/json;charmer=UtF-8,
            url:xxx,
            data:Jason.dumps,
            success:function(res),
            error:function(e)
        }
        ```
